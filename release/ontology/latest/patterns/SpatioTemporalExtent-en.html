Note

This template defines the pattern metadata. The actual pattern implementation uses OWL-Time and GeoSPARQL vocabularies.

Description

Pattern for modeling unified spatial and temporal extents of events and entities.

Classes

Spatiotemporal Extent c

IRI https://vocab.earth616.localhost/ont/SpatioTemporalExtent
Is Defined By Spatiotemporal Extent Pattern
Description

  • A unified spatial and temporal extent that combines both geometric and temporal aspects. Extends W3C Time and OGC GeoSPARQL vocabularies for supply chain domain modeling.

  • The combined spatial and temporal boundaries of an event or process
Example 
A transportation event that occurs from warehouse A to warehouse B between 09:00 and 11:30 on 2024-03-15
Sub Class Of
  • time:TemporalEntity
  • geo:SpatialObject
